How BFS Works

Breadth-First Search explores all puzzle states level by level. It uses a queue (FIFO) and guarantees finding the shortest solution path. However, it explores exponentially many states — for an 8-puzzle, this can be thousands of nodes. It doesn't use any domain knowledge.

How A* Works

A* uses a priority queue ordered by f(n) = g(n) + h(n) where g(n) is the cost so far and h(n) is the Manhattan distance heuristic. It intelligently focuses the search toward the goal, exploring far fewer nodes than BFS while still guaranteeing an optimal solution.

The ML Component

A Linear Regression model is trained on features extracted from puzzles: Manhattan distance sum, number of misplaced tiles. The model learns the relationship between these features and the actual number of steps required to solve. R² score measures fit quality; a score near 1.0 means near-perfect prediction.
